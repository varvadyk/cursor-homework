• git config - global user.name blabla - оприділення ім'я користувача
  • git config - global user.email blabal@bla - оприділення почти користувача
  • git init - створення нового репозиторію
  • git status - повертає інформацію про поточний стан репозиторія: що нового, що змінилося
  • git add - додавання файлів до репозиторію
  • git commit - зліпок поточних файлів з їх змінами
  • git log - відображає історію коммітів для поточної гілки
  • git checkout - перемикається на обрану гілку і оновлює робочу директорію до її стану
  • git branch - список іменованих гілок коммітів з вказанням обраної гілки
  • git pull - отримання змін з серверу,завантажує історію з віддаленого сховища та об'єднує її з локальної
  • git merge - використовується для злиття однієї або декількох гілок в поточну
  • git clone - cкачує репозиторій разом з усією його історією змін

Gitignore потрібен для приховування файлів і папок від системи контролю версій Git. Зазвичай приховують конфігураційні файли (особливо з паролями), тимчасові файли і папки. gitignore використовує glob формат для вибірки файлів.

Основний синтаксис:

Кожен рядок - окремий шаблон
Порожні рядки ігноруються
Рядки починаються з # є коментарями
Символ слеша "/" на початку рядка вказує на поточну папку (де лежить .gitignore)
Зірочка (*) замінює будь-яку кількість символів
Дві зірочки (**) використовуються для вказівки всіх підпапок.
Знак оклику (!) На початку рядка інвертує шаблон (використовується для винятків)
Для екранування спецсимволов використовується зворотний слеш "\".

  • git revert - бере всі зміни, які були у коміті, відміняє їх, і створює новий комміт.
  • git reset - cкасовує всі комміти після заданого, залишаючи всі зміни в робочій директорії
  • git cherry-pick - використовується для того щоб взяти зміни, внесені будь-яким коммітов, і спробувати застосувати їх заново у вигляді нового коммітов нагорі поточної гілки.
  • git rebase - один з двох способів об'єднати зміни, зроблені в одній гілці, з іншого гілкою
  • git stash - тимчасово зберігає всі незафіксовані зміни
